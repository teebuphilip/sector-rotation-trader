# TL;DR Tutorial

## What this is

StockArithm is a public paper-trading lab for alternative data signals.

## Start here

1. Home = short explanation.
2. Leaderboard = live scorecard.
3. Signals = full inventory.
4. Families = signal groups.
5. Daily Report = last nightly update.

## Important terms

- SPY = market benchmark
- Drawdown = worst drop from a peak
- Sharpe = rough return vs risk measure
- Live-only = public signal, not fully backfilled
- Promoted = stronger public signal
- Parked = inactive
- Graveyard = dead end kept visible

## What to expect

- Some signals work.
- Some fail.
- Failures stay visible.
- The public site does not give away the full ranking edge.

## One-line summary

It is a public lab, not a polished stock-pick product.
